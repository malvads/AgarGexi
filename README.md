This was my first extension that I made for the Agar.io Game, it was originally developed on 2016 when I was 13, code was recovered from https://pastebin.com/raw/5N2Fkv3r and DeObfuscated using https://obf-io.deobfuscate.io/ and GPT-5.3 (thinking)

You can see what it used to look here https://www.youtube.com/watch?v=TmD_Iefv3rY

## Characteristics

- Built as a browser-based Agar.io extension / mod.
- Heavy DOM manipulation and UI injection.
- Uses canvas interception to read leaderboard data.
- Persists settings with `localStorage`.
- Adds custom overlays such as minimap, sectors, borders, and debug labels.
- Includes skin management features, including custom skins and skin rotation.
- Adds gameplay toggles such as zoom, grid visibility, minimap visibility, stroke text, and theme options.
- Includes macro / keybind support for split and eject actions.
- Contains mini-game / challenge-style timers and status messages.
- Integrates with external scripts and game APIs at runtime.
- Reflects an early experimental reverse-engineered approach rather than a polished production architecture.

## Main Features

- Custom minimap with sector labels
- Map borders and grid overlays
- Custom skin support
- Skin changer / skin rotation
- Hide names, skins, colors, and mass display options
- Dark / black theme options
- Zoom customization
- Macro keys for split and eject
- Match / survival challenge timers
- FPS counter and debug HUD
- Saved settings across sessions

## Notes

This project is preserved as a historical snapshot of an early modding experiment. Some parts are incomplete, experimental, or tied to older external services that may no longer work.
